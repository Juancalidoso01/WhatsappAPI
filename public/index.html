      Crear plantilla

      Tras crearla, Meta la revisa (minutos a horas). Si usas variables ({{1}}, {{2}}…), define claves para la API y ejemplos — Meta los exige al crear la plantilla.

      Reglas de Meta para variables · evita rechazos
        Variables / placeholders

        Solo si usas {{1}}, {{2}}… en el cuerpo. Clave = nombre en la API; ejemplo = valor que Meta revisa al aprobar. Usa la biblioteca de variables en Plantillas como referencia.
